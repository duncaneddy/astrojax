# API Reference

Complete reference for all public modules in astrojax.

| Module | Description |
|--------|-------------|
| [Config](config.md) | Module-wide float precision configuration |
| [Constants](constants.md) | Physical and mathematical constants used throughout the library |
| [Time](time.md) | Calendar date, Julian Date, and Modified Julian Date conversions |
| [Epoch](epoch.md) | High-precision time representation with JAX pytree support |
| [Frames](frames.md) | ECI-ECEF frame transformations using Earth rotation |
| [Coordinates](coordinates/index.md) | Geocentric, geodetic, Keplerian, and ENZ topocentric transformations |
| [Orbits](orbits/index.md) | Orbital mechanics, anomaly conversions, and mean-osculating element transformations |
| [Orbit Dynamics](orbit_dynamics/index.md) | Perturbation force models (gravity, drag, SRP, third-body) |
| [Relative Motion](relative_motion/index.md) | ECI-RTN transformations, Relative Orbital Elements (ROE), and HCW dynamics |
| [Attitude Representations](attitude_representations.md) | Quaternion, rotation matrix, Euler angle, and Euler axis classes |
| [Attitude Dynamics](attitude_dynamics.md) | Rigid-body attitude dynamics (Euler equations, gravity gradient) |
| [Integrators](integrators.md) | Numerical integration methods |
